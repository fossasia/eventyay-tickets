Welcome to the documentation of the eventyay tickets component!

We work hard to make this website contain all information that you need to use, run, understand, and improve eventyay. Of course, this documentation will never be perfect or complete, but if there is anything unclear or anything specific that you miss here, that's a bug and we'd be happy if you'd let us know.

Documentation structure

User Guide

Go here to find information on how to configure and use the platform as an event organizer.

Administrator docs

Find out how to install eventyay on your own server and how to maintain an installation of eventyay.

REST API

Documentation and reference of the RESTful API exposed by eventyay for interaction with external components.

Developer docs

Get information on how to contribute to eventyay itself and how to build plugins that interact with eventyay.

Plugin docs

Documentation and details on plugins that ship with the platform or are officially supported.
